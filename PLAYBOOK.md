# PLAYBOOK.md

**Projeto:** FaculdadeMaria\
**Documento:** Playbook Oficial de Desenvolvimento\
**Versão:** 1.0

------------------------------------------------------------------------

# Princípios

O FaculdadeMaria é um sistema para apoiar a venda sistemática de PUTs na
B3.

O objetivo NÃO é buscar o maior prêmio.

O objetivo é encontrar oportunidades seguras para gerar renda e adquirir
ativos de qualidade.

------------------------------------------------------------------------

# Filosofia

-   O exercício faz parte da estratégia.
-   Qualidade do ativo vem antes do prêmio.
-   O ROI alvo oficial é **4%**.
-   O Decision Engine é o cérebro do sistema.
-   Toda recomendação deve ser explicável.

------------------------------------------------------------------------

# Como a IA deve agir

-   Responder de forma objetiva e rápida.
-   Discordar do usuário quando houver fundamento técnico.
-   Nunca recomendar uma operação apenas pelo prêmio.
-   Explicar o motivo da recomendação em poucas linhas.

------------------------------------------------------------------------

# Ordem de Prioridade

1.  Qualidade do ativo
2.  Segurança
3.  Preço líquido
4.  Eficiência do capital
5.  Liquidez
6.  Probabilidade de exercício
7.  ROI
8.  Prêmio

------------------------------------------------------------------------

# Regras do Radar

-   Buscar oportunidades novas.
-   Nunca listar operações abertas.
-   Exibir Score IA, ROI, preço líquido, prêmio e resumo técnico.

------------------------------------------------------------------------

# Padrão de Desenvolvimento

-   Trabalhar por Sprints.
-   Nunca alterar a `main` diretamente.
-   Atualizar documentação.
-   Executar testes antes do merge.
-   Merge apenas com autorização do Product Owner.

------------------------------------------------------------------------

# Experiência do Usuário

A interface deve transmitir sensação de software Premium:

-   limpa;
-   elegante;
-   rápida;
-   intuitiva;
-   com boa legibilidade.

------------------------------------------------------------------------

# Visão de Longo Prazo

Transformar o FaculdadeMaria em uma plataforma profissional para
análise, gestão e acompanhamento de operações com opções da B3, com IA
explicável e foco em segurança, consistência e qualidade das
recomendações.
